# Paper Readiness Checklist

## Completed Non-Expert Components

- [x] Real semantic embedding path.
- [x] BM25 sparse retrieval.
- [x] Hybrid retrieval.
- [x] Cross-encoder reranker.
- [x] Formal ablation matrix.
- [x] Public answer-target benchmark.
- [x] Expanded hard OOD tests.
- [x] Table/figure metadata benchmark.
- [x] Cell-level table QA benchmark.
- [x] Realistic agent-task evaluation.
- [x] Answer-quality proxy evaluation.
- [x] Failure taxonomy.
- [x] Public release boundary audit script.
- [x] GitHub/Zenodo metadata files.
- [x] Academic norm review boundary.

## Remaining Before Final Article Submission

- [ ] Write the full manuscript.
- [ ] Generate final publication figures.
- [ ] Create a GitHub release tag.
- [ ] Mint or attach a Zenodo DOI.
- [ ] Decide whether expert review is available.
- [ ] If expert review is available, run expert answer/citation scoring.
- [ ] If expert review is not available, keep claims at public-source
      engineering validation level.

## Suggested Minimum Manuscript Results

- Retrieval strategy table.
- Formal ablation table.
- Agent task table.
- Table-cell and asset QA table.
- Answer-target and answer-generation comparison table.
- Failure taxonomy table.
- Conservative limitations section.
